# SlamBook [https://github.com/VikasMeshram2708/Slambook-2.0-deploy]

### Setup

* git clone https://github.com/VikasMeshram2708/Slambook-2.0-deploy

* cd Slambook-2.0-deploy

* npm install

* npm run dev
